# Changelog

## 1.0.0 — 2024-05-05

First release.

- Menu bar clipboard manager
- Auto text monitoring (0.5s poll)
- SQLite local storage, 1–30 day retention
- Auto classify: URL, email, code, number, Chinese, English, mixed
- AI summarization via Ollama
- AI custom categories with prompt
- Full-text search + category sidebar
- Favorites (won't get cleaned up)
- Global hotkey ⌘⇧V, customizable
- Settings: retention, hotkey, AI connection, custom categories, API history
- Left-click popover, right-click context menu
