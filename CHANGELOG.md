# Changelog

## [1.0.0] — 2024-05-05

### Added
- Menu bar clipboard manager with popover UI
- Automatic text clipboard monitoring (0.5s polling)
- SQLite-based persistent storage with configurable retention (1–30 days)
- Built-in content categorization: URL, Email, Code, Number, Chinese, English, Mixed
- AI summarization via local Ollama
- AI custom categories with prompt-based classification
- Full-text search and category sidebar filtering
- Favorites system (starred items excluded from auto-cleanup)
- Global hotkey (default ⌘⇧V) with customizable key recording
- Settings window: retention slider, hotkey config, AI connection, custom categories, API usage history
- AppKit status bar item with left-click popover and right-click context menu
